# Effects

Lost Tales does not currently provide a general custom status-effect framework.

The One Ring has dedicated accessory effects: a wearer-only wraith-world visual and server-owned public concealment. Quest notifications, camera feedback, charge-tier particles and sounds, and race attributes are feature-specific behavior rather than entries in a shared effect system.

See [Accessories and the One Ring](Accessories_and_the_One_Ring.md) for the implemented behavior.
